A.compareTo(B)
A.compareToIgnoreCase(B) - 대소문자 무시
 -> A가 B보다 사전상 앞에 있으면 음수
 -> 뒤에 있으면 양수
 -> 같으면 0


a.contains(b) : a가 b를 포함하고 있는지
a.startsWith(b) : a가 b로 시작되는지
